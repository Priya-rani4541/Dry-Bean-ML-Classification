# Dry Bean Classification using Machine Learning

This project applies supervised and unsupervised ML techniques to classify dry bean varieties using geometric features.

## Dataset
13,000+ samples | 16 features | 7 classes

## Models Used
Logistic Regression, Random Forest, SVM, MLP

## Unsupervised Learning
K-Means Clustering, Hierarchical Clustering

## Dimensionality Reduction
PCA

## Result
MLP achieved ~92% accuracy.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
